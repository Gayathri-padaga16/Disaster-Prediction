 Disaster Prediction

 Overview
Disaster Prediction is an AI-powered application that helps predict potential disasters using data analysis and machine learning techniques.

 Features
- Data preprocessing and cleaning
- Disaster prediction using machine learning
- Weather data integration
- Automated prediction workflow
- User-friendly interface

 Technologies Used
- Python
- Machine Learning
- CSS
- OpenWeather API

Project Files
- app.py – Main application
- auto_predict.py – Prediction module
- clean_dataset.py – Data cleaning
- excel_to_csv.py – Data conversion utility
- animations.css – Frontend styling

 How to Run

1. Clone the repository

```bash
git clone https://github.com/Gayathri-padaga16/Disaster-Prediction.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file and add your OpenWeather API key.

4. Run the application

```bash
python app.py
```
